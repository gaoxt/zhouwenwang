# 周文王占卜系统使用说明

## 🚀 快速开始

### 第一次使用

1. **获取 API 密钥**
   - 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)
   - 用 Google 账号登录
   - 点击"创建API密钥"获取密钥（格式类似：AIzaSyC...）

2. **配置密钥**
   - 打开应用设置页面（右上角设置图标）
   - 在"API配置"中粘贴您的密钥
   - 点击保存

3. **开始占卜**
   - 选择占卜方式：六爻、奇门遁甲或手相
   - 选择您喜欢的大师
   - 输入问题，开始占卜

## 📖 功能介绍

### 🔮 六爻占卜
**最经典的占卜方式，通过摇卦获得指引**

使用步骤：
1. 点击左侧"六爻占卜"
2. 在文本框输入您想问的问题（如：我的工作运势如何？）
3. 点击"开始摇卦"观看摇卦动画
4. 选择一位大师为您解卦
5. 阅读详细的卦象分析

适合问题：
- 事业发展方向
- 感情婚姻状况
- 财运投资建议
- 健康状况询问
- 人际关系处理

### ⭐ 奇门遁甲
**古代最高层次的预测学，适合重大决策**

使用步骤：
1. 点击"奇门遁甲"
2. 输入要预测的具体事项
3. 确认起局时间（默认当前时间）
4. 选择擅长奇门的大师（推荐诸葛亮）
5. 获得深度的局势分析

适合问题：
- 重要投资决策
- 搬家择日
- 商业合作选择
- 职业规划
- 时机把握

### ✋ 手相分析
**通过手掌纹路解读性格和命运**

使用步骤：
1. 点击"手相分析"
2. 点击"上传照片"选择手掌照片
3. 确保照片清晰，手掌完整可见
4. 选择精通相术的大师（推荐陈图南）
5. 获得详细的手相解读

拍照技巧：
- 使用自然光，避免阴影
- 手掌平铺，五指自然伸展
- 照片清晰，掌纹可见
- 建议拍摄主导手（写字的手）

## 💡 使用技巧

### 提问技巧
- **具体明确**：避免"我运势怎么样"，改为"我近期的工作运势如何"
- **单一问题**：一次只问一个主要问题
- **时间明确**：可以指定时间范围，如"未来三个月"

### 问题示例
- ✅ 好问题：我是否应该接受这个新工作机会？
- ❌ 模糊问题：我的命运如何？
- ✅ 好问题：我和某某的感情关系会如何发展？
- ❌ 模糊问题：我的爱情运势？

### 最佳实践
- 保持开放的心态
- 将结果作为参考，不要完全依赖
- 关注建议中的积极指导
- 定期回顾历史记录，总结经验

## 📚 历史记录

### 查看历史
- 点击右上角"历史"图标
- 按时间倒序查看所有占卜记录
- 点击任意记录查看详情

### 管理历史
- 支持按占卜类型筛选
- 可以删除不需要的记录
- 重要记录可以添加备注

## ⚙️ 设置选项

### 界面设置
- **侧边栏**：可以收起/展开
- **主题**：目前支持暗色主题
- **动画**：可以关闭摇卦动画

### API 配置
- **Gemini密钥**：输入您的API密钥
- **模型选择**：选择AI模型（默认即可）
- **响应设置**：调整响应速度和详细程度

- 如果密钥泄露，请立即重新生成

### 使用建议

### 常见问题

**Q: 显示"API密钥无效"怎么办？**
A: 检查密钥是否正确复制，确保以"AIza"开头

**Q: 分析结果不准确？**
A: 尝试更换大师或重新表述问题

**Q: 手相识别失败？**
A: 确保照片清晰，光线充足，手掌完整

**Q: 响应速度慢？**
A: 检查网络连接，AI分析需要一定时间
